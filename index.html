Vedic Accents & Special Marks
How to type Vedic accents with your physical keyboard:
  • Type _ (underscore) for Anudatta (ಕ॒)
  • Type + (plus) for Swarita (ಕ॑)
  • Type ++ (two pluses) for Dirgha Swarita (ಕ᳚)
  • Type M (capital M) for Anusvara (ಕಂ)
  • Type H (capital H) for Visarga (ಕಃ)
  • Type MM (two capital M's) for Candrabindu (ಕಁ)

Typing Guide

Vowels & Specials
  • Long: type twice or use caps (aa / A = ಆ)
  • ai = ಐ | au = ಔ
  • -r = ಋ | -l = ಌ
  • M = Anusvara (ಂ)
  • MM = Candrabindu (ಁ)
  • H = Visarga (ಃ)
  • ' = Avagraha (ಽ) | = = Nukta (಼)
Consonants
  • Caps for retroflex: T(ಟ್), D(ಡ್), N(ಣ್), L(ಳ್)
  • Add h for aspirated: kh (ಖ್), bh (ಭ್)
  • G or ng = ಙ್ | J = ಞ್
  • sh/z/ç = ಶ್ | S = ಷ್ | s = ಸ್
  • r= = ಱ್ | l= = ೞ್
Conjuncts & Clusters
  • kSa = ಕ್ಷ | jJa = ಜ್ಞ
  • tra = ತ್ರ | shra = ಶ್ರ
  • Other ottakshara: type back-to-back (kka = ಕ್ಕ)
  • x = zero-width joiner (explicit half form)
  • xx = prevent conjunct forms
Vedic Accents
  • _ for Anudatta (ಕ॒)
  • + for Swarita (ಕ॑)
  • ++ for Dirgha Swarita (ಕ᳚)
  • / or | for Danda (।), twice for ॥
